Prerequisites & Prework

Prepare yourself for success in this course

Is this course right for you?

Machine Learning Crash Course does not presume or require any prior knowledge in machine learning. However, to understand the concepts presented and complete the exercises, we recommend meeting the following prerequisites:

Prework

+

Before beginning, complete the following:

  1. Introduction to Machine Learning

    If you're new to ML, take the intro course to learn fundamental concepts

    Visit Course
  2. NumPy Tutorial

    Complete the NumPy Ultraquick Tutorial for essential array operations

    NumPy Website Ultraquick Tutorial
  3. pandas Tutorial

    Complete the pandas UltraQuick Tutorial for data manipulation skills

    pandas Website Ultraquick Tutorial

Mathematics

+

Algebra

  • Variables, coefficients, and functions
  • Linear equations (e.g., y = b + w₁x₁ + w₂x₂)
  • Logarithms and logarithmic equations
  • Sigmoid function

Linear Algebra

  • Tensor and tensor rank
  • Matrix multiplication

Statistics

  • Mean, median, outliers, standard deviation
  • Ability to read histograms

Calculus (Optional)

  • Concept of derivatives
  • Gradient or slope
  • Partial derivatives and chain rule

Programming

+

Python Basics

  • Defining and calling functions
  • Dictionaries, lists, and sets
  • For loops and if/else blocks
  • String formatting
  • Basic data types (int, float, bool, str)
  • List comprehensions

Command Line (Optional)

  • Basic Bash terminal commands
  • Working with cloud consoles

Experience Level

+

✅ Perfect if you:

  • Have little or no ML background
  • Want a current understanding of ML
  • Are comfortable with algebra and Python
  • Want to build practical ML skills

About This Course

This is a simplified version of Google's Machine Learning Crash Course. Since 2018, millions of people worldwide have relied on this course to learn how machine learning works and how it can work for them.

Each module is self-contained, so if you have prior experience in machine learning, you can skip directly to the topics you want to learn. If you're new to machine learning, we recommend completing modules in the order presented.
